# MathWorld
# Description: 
MathWorld is an educational C++/Qt application designed to help students practice core calculus concepts through an interactive game.
It features question banks for derivatives, integrals, and limits, a formula sheet reference, and automatic explanation feedback to support learning.

# Features: 
. Random question selection
. Derivative, Integral, and Limit modules
. File loading using text files
. Step-by-step explanations
. Clean Qt GUI with stacked widgets
. Object-oriented class structure with inheritance

# Technologies
. C++
. Qt Creator (Widgets)
. MinGW Compiler
. OOP (inheritance, polymorphism)
. File I/O
. Random number generator (<random>)
